# cis4340-lab02-github
CIS4340 Lab 02 - Java security vulnerabilities from the SEI CERT Oracle Coding Standard with GitHub tracking of fixes.

This repository contains 10 short Java programs demonstrating security vulnerabilities and their compliant solutions based on the SEI CERT Oracle Coding Standard for Java.

Contents:
• 5 mandatory vulnerabilities required by the assignment
• 5 additional vulnerabilities selected from different rule sets
• Each file contains noncompliant and compliant code examples
• GitHub commit history tracks the applied fixes
